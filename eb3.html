LilyPad Button Board Module

The LilyPad Button Board Module is part of the LilyPad Arduino collection, which is a set of sewable electronic modules designed for wearable and e-textile projects. These modules are designed to be connected with conductive thread, making them perfect for integrating into fabric-based projects.

Simple Momentary Push Button:

Acts as a digital input.

Normally open contact.

Designed for Wearables:

Sewable pads for easy integration into textiles.
Designed to be connected with conductive thread.

Compact and Lightweight:

Small form factor suitable for embedding into clothing and accessories.
Lightweight to ensure comfort when worn.

Wearable Electronics:

Control elements in smart clothing.
Input device for interactive garments.

Educational Projects:

Great for teaching electronics and programming in a creative context.
